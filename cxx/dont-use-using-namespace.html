Don't use using namespace

Each programming tutorial starts with the famouse "Hello World!" example. If you type "c++ hello world" into the Google you will probably find something like the following code: 

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello, World!" << endl;
    return 0;
}

This introduces a bad practice of importing whole std namespace into global scope.
